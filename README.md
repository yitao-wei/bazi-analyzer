# bazi-analyzer

微信小程序 · 生辰八字简易分析工具

A simple WeChat Mini Program for generating Bazi (Four Pillars of Destiny) analysis based on birth date, birth place and gender.

## 功能

- 输入出生日期、出生地、性别
- 自动计算生辰八字
- 生成简洁易懂的命理分析文案
- 轻量、无后端、纯前端实现

## 技术栈

- 微信小程序原生开发
- JavaScript / WXML / WXSS
- 八字历法计算逻辑

## 使用方式

1. 克隆本项目
2. 使用微信开发者工具打开项目
3. 填入 AppID 即可运行

## 说明

本项目仅为传统文化趣味分析工具，结果仅供娱乐参考，非专业命理咨询。
