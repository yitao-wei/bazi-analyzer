# 知命生辰产品技术设计文档（Tech Design）

## 1. 文档概述

### 1.1 文档目的
基于《知命生辰产品需求文档（PRD）》，明确 **MVP（v1.0）** 基础功能的技术实现方案，为微信小程序原生开发（`wxml`/`wxss`/`js`）、云开发提供统一规范，保障开发落地与交付质量。

### 1.2 适用范围
适用于本产品微信小程序前端开发、后端云函数开发、测试人员，作为开发、联调、测试的核心依据。

### 1.3 产品核心定位
**新中式美学 + 现代性格分析** 的轻量化命理工具，通过微信原生开发实现简洁交互，完成八字排盘、五行计算及简易分析报告生成。

## 2. 整体架构设计

### 2.1 架构选型
采用 **“微信原生前端 + 微信 CloudBase 云开发”** 轻量化架构，无需自建服务器，降低开发成本，适配小程序场景，分为 3 层：

- **前端交互层**：微信小程序原生开发（`wxml`/`wxss`/`js`），负责用户输入、界面渲染、本地基础存储。
- **云函数服务层**：微信 CloudBase 云函数，部署核心计算逻辑，提供接口供前端调用。
- **数据存储层**：微信小程序本地存储（`storage`）+ CloudBase 云数据库，分别存储用户查询历史与基础配置数据。

### 2.2 核心技术依赖

| 技术类型 | 具体依赖 | 用途说明 |
| :--- | :--- | :--- |
| **前端开发** | 微信小程序原生（`wxml`/`wxss`/`js`） | 页面渲染、用户交互、本地存储、可视化展示 |
| **后端服务** | 微信 CloudBase（云函数、云数据库） | 核心计算、配置数据存储与管理 |
| **历法计算** | `lunar-javascript` 开源库 | 公农历转换、干支排盘 |
| **可视化** | 微信小程序原生画布 | 五行占比环形图/雷达图渲染 |

## 3. 核心模块技术实现（MVP v1.0）

### 3.1 用户输入模块

**功能实现**  
实现公农历切换、出生时间（精确到时）输入、城市级出生地点选择、性别选择，完成输入校验并提交参数。

**技术细节（微信原生开发）**
- **公农历切换**：基于 `lunar-javascript` 库，实现公农历双向切换，同步日期，确保参数一致。
- **出生时间选择**：使用小程序原生 `picker` 组件（日期 + 时间），精确到小时，默认当前日期，限制近 100 年范围。
- **出生地点选择**：内置省 - 市二级联动城市列表（云数据库存储，前端启动异步加载），记录城市经纬度（预留）。
- **性别选择**：`radio` 组件（男/女），存储标识（`1=男 `，`2=女`），为后续迭代预留。
- **输入校验**：前端 `js` 编写校验逻辑，未补全必填项时提示用户，校验通过后格式化参数，调用云函数接口。

### 3.2 核心计算引擎（云函数）

**功能实现**  
完成干支排盘、五行强弱量化、十神画像计算，返回前端所需数据。

**技术细节**
- **云函数定义**：创建 `calcBazi` 云函数，接收前端传入参数（出生时间戳、公农历标识、经纬度、性别），返回计算结果。
- **干支排盘**：通过 `lunar-javascript` 转换时间，结合“五虎遁”“五鼠遁”规则，计算年、月、日、时四柱干支及对应五行。
- **五行强弱**：定义干支与五行映射关系，量化各五行能量分值，计算百分比及强弱状态（阈值：`<10%` 偏弱，`>30%` 偏强）。
- **十神画像**：基于日柱天干，匹配十神定义，统计主导十神，为性格标签提供支撑。
- **性能优化**：基础映射关系云数据库配置，云函数启动时加载至内存，运算时间控制在 `300ms` 内。

### 3.3 分析文案模块

**功能实现**  
生成 3-5 个性格标签、3-4 条五行生活化建议，贴合新中式风格，避免深奥术语。

**技术细节**
- **文案配置**：云数据库创建性格标签表、五行建议表，存储对应文案。
- **性格标签**：根据主导十神匹配标签，筛选 3-5 个不重复标签。
- **五行建议**：根据五行强弱结果，匹配生活化建议（颜色、方位等）。
- **前端渲染**：标签云展示性格标签，卡片展示五行建议，视觉贴合新中式（宣纸白、水墨灰、朱砂红）。

## 4. 非功能性需求落地

### 4.1 性能要求
- **首页冷启动**：速度控制在 `1.2s` 以内，优化小程序包体积，减少冗余资源。
- **核心计算逻辑**：云函数运算时间 `≤300ms`，确保用户体验。

### 4.2 安全与隐私
- **数据存储**：用户出生信息仅本地存储或加密处理，不关联敏感身份信息，不随意上传。
- **合规提示**：前端页面显著位置添加合规提示：“本工具仅供娱乐与传统文化参考，请相信科学”。

### 4.3 审核避坑
- **文案规范**：禁用“算命、改命、化解劫难”等违规字眼，统一使用“分析、参考”等合规表述。
- **类目选择**：小程序类目选择 **“生活服务 > 工具”** 或 **“教育 > 传统文化”**。
